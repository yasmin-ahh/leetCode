# LeetCode Solutions

A collection of solutions to LeetCode problems, written with a focus on clean, 
readable code and efficient approaches. Each folder contains a problem, my 
solution, and some notes on the approach and complexity.

## Structure

Each problem lives in its own folder containing the solution and a short 
explanation of the approach and time/space complexity.

## Topics Covered

- **Arrays & Strings** — Two Sum, Rotate Array, Group Anagrams, Valid Anagram
- **Hashing** — Count Unique Characters, Majority Elements
- **Trees** — Symmetric Tree, Validate BST
- **Dynamic Programming** — Climbing Stairs, Max Sum Path
- **Design** — LRU Cache
- **Math** — Integer to Roman, String to Integer, Count Primes, FizzBuzz

## Language

Solutions are written in [C++ / Python].
